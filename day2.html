Cascading Style Sheets - CSS

Key - value pairs Ex: color : red

Css can be included in three locations

  1. In HTML Element
  2. In HEAD HTML Element by creating STYLE Tag
  3. In Separate CSS File and Add the reference in HTML

Different Types of Selectors

  1. Selecting an element using Tag names. Ex: h1, h2, p, span, div, li ... etc
  2. Id selector #
  3. Class selector .
  4. Attribute Selector [attributeName= "attributeValue"]
  5. Universal Selector *
  6. Group selector
  7. Child Selector (>)
  8. Descendent selector (space)
  9. Adjacent Sibling selector (+)
  10. Sibling Selector (~)
